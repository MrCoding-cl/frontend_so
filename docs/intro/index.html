Instalaci贸nes#

Frontend#

Si desea instalar y correr localmente el Frontend, debera tener instalado Nodejs:

Despues de eso es tan simple como correr el siguiente comando

npm i

Lanzamiento del sitio web#

Se corre localmente:

cd frontend
npm run start

El sitio se lanza en el puerto http://localhost:3000.

Backend#

Para instalar e iniciar el Backend de manera local, debe de tener instalado una version de Golang igual o superior a la version v1.16 luego correr el siguiente comando

go get -u ./

en el directorio donde ha clonado el repositorio de GitHub

Lanzamiento del servidor#

Una vez suplidas las dependencias (Fiber). Posee dos opciones para correr el programa

# 1 Generar un exe
go build
# 2 Correr el codigo
go run ./

Si eligio la primera opcion, debe de buscar el exe generado en la carpeta y arrancarlo como un ejecutable normal.

El servidor se lanzara en el puerto 8080
